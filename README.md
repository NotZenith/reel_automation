# 🎬 Reel Automation

A simple Python automation tool that helps generate captions, hashtags, and manage reel posting workflow.

## ⚡ What it does

- 🎥 Scans a folder for video files
- 🧠 Generates random captions
- 🏷️ Suggests relevant hashtags
- 📊 Logs all generated data into a CSV file
- ⚙️ Helps automate content preparation workflow

## 📁 Supported Formats

- `.mp4`
- `.mov`
- `.mkv`

## 🚀 How to use

```bash
python reel_automation.py# reel_automation
